Docker 环境配置说明 
⚠️ 闪传功能需要 AstrBot 和 NapCat 共享文件目录。

如果你是 Docker 环境，请按以下步骤配置：

1. 创建共享目录（宿主机）
mkdir -p /opt/astrbot_flash

2. AstrBot 容器添加挂载
docker run -v /opt/astrbot_flash:/tmp/astrbot_flash:rw ...

3. NapCat 容器添加挂载
docker run -v /opt/astrbot_flash:/tmp/astrbot_flash:ro ...

4. 在上方填入容器内路径: /tmp/astrbot_flash

如果不是 Docker 环境，可以忽略此配置。
